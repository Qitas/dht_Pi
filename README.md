# [sensors4Pi](https://github.com/Qitas/sensors4Pi) 

#### qitas@qitas.cn

## [简介](https://github.com/Qitas/sensors4Pi/wiki)

本项目主要编辑和收集运行在各个Pi上的传感器驱动和应用，主要包括树莓派的相关传感器驱动代码
